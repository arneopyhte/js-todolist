# js-todolist
This is project created using javascript.I used the browser local storage to store the list and user's name.

## Technologies
Project is created with:
* Html5
* Javascript
* CSS
* Bootstrap 5 : 5.0.2
* Bootstrap icons: 1.8.3
* Local storage

### Funtionality
* Add list
* Remove list
* Update list
* Clear all list
* Counts remaining task
* Remove all completed task
